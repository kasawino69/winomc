# Security Policy

## Supported Versions

Security updates are provided for the currently maintained versions of WinoMC.

| Version | Supported          |
| ------- | ------------------ |
| > 1.6.0 | :white_check_mark: |
| < 1.6.0 | :x:                |

Older versions are not actively maintained and may not receive security fixes.
Please update to the latest available version before reporting issues.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening a GitHub Issue in this repository.

When creating the issue, please include as much helpful information as possible:

* A clear description of the vulnerability
* Affected version
* Steps to reproduce the problem
* Expected behavior
* Actual behavior
* Relevant logs, configuration snippets, or screenshots if helpful

Please avoid posting sensitive information such as tokens, passwords, private server addresses, or personal data in public issues.

We will review reported vulnerabilities as soon as possible.
If the report is accepted, we will work on a fix and provide an update through the related issue, release notes, or changelog.

If the report is declined, we will explain the reason in the issue when possible.

Thank you for helping improve the security of WinoMC.
